#Dotfiles
##Install
To install my dotfiles, run the following:

    ruby -e "$(curl -fsSL https://raw.githubusercontent.com/jelmersnoeck/dotfiles/master/install/remote-install)" && bash /tmp/install-dotfiles

## Apps
To install default apps:

`./.brew`
